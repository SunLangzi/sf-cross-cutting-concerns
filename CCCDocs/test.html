Common

The Common classes are made up of helpers, constants, utilities, etc. There are some aspects, such as IHttpUtility, which are design to provide the ability to respond/send information over Http. Which would fall under the Communication layer. However, I have not gotten that far. The main focus was logging, exception handling, caching and the ability to extend or change. Similar to more robust Object Orient Languages like Java, C#, etc. which support reflection, closure, etc.
